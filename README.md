# ⚖️ Legal AI Assistant (Agentic AI Project)

## 📌 Overview
This project demonstrates an **Agentic AI system** in the legal domain using Python.  
It simulates how multiple AI agents collaborate to process and respond to legal queries.

---

## 🎯 Objective
To design and implement a modular AI system that:
- Accepts legal queries
- Searches relevant legal data
- Performs reasoning
- Generates structured responses

---

## 🧠 Agent Architecture

Input Agent → Research Agent → Reasoning Agent → Response Agent

---

## ⚙️ Features

- ✅ Menu-based CLI interface
- ✅ Streamlit Web Application
- ✅ AI-powered legal responses
- ✅ Modular agent-based design
- ✅ JSON-based legal knowledge base

---

## 🛠️ Tech Stack

- Python
- Streamlit
- OpenAI API
- JSON
